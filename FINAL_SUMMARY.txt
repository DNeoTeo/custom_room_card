🎯 UTILISATION RAPIDE
────────────────────────────────────────────────────────────────────────────

1. INSTALLATION VIA HACS (Recommandé)
   - Allez dans HACS → Lovelace
   - Cherchez "Custom Room Card"
   - Installez et redémarrez Home Assistant

2. CONFIGURATION SIMPLE
   type: custom:custom-room-card
   title: "Ma Pièce"
   background_color: "#f5f5f5"
   height: "400px"
   entities:
     - entity: light.salon
       label: "Lumière"
       top: "50%"
       left: "50%"

3. AVEC IMAGE DE FOND
   type: custom:custom-room-card
   background_image: /local/salon.jpg
   background_opacity: 0.85
   height: "450px"
   entities:
     - entity: light.salon
       label: "Lumière"
       top: "30%"
       left: "30%"

4. MODE DÉVELOPPEMENT (GRILLE)
   type: custom:custom-room-card
   show_grid: true
   grid_size: 50
   height: "500px"
